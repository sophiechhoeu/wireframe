WIREFRAME: ASOS

I've picked asos for a redesign because their current website style is cramped and generic. The colours they've chosen are bright but not subtle. It bombards the user on the homepage.

AXURE

I used Axure to build out my wireframe for asos which included a hidden menu and searchbar. By doing this we've kept the features of the shopping website without crowding the homepage. The wireframe suggests 2 images instead of one large image above the fold to aid user experience. You can check out my wireframe: here.

PINTEREST MOODBOARD

The pinterest moodboard showcases muted pastel colours. The moodboard also demonstrates how bright colour can be used in a subtle way. This aesthetic is minimialistic,millenial and on trend.You can check out the pinterest board: here.
